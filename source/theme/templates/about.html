About Matt Makai

I'm a full stack software developer based in our nation's capital. From March through August 2013, I traveled to 30 cities across the United States to talk with software developers as part of my Coding Across America self-defined project.

In January 2014 I will start at Twilio as their Washington, D.C.-based Developer Evangelist. My job is to inspire and equip software developers so they can build amazing new communications applications.

My primarily technology skill set is in the Python ecosystem. My secondary programming capabilities are in the JVM stack with Java and Clojure. I also create data visualizations, primarily with d3.js and I'm beginning to get into data analysis with pandas, numpy, scipy, matplotlib, and iPython Notebook.
